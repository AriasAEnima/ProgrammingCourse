## 📦 Instalación de Django

```bash
# Crear entorno virtual
python -m venv django_env

# Activar entorno virtual
# En Windows:
django_env\Scripts\activate
# En Mac/Linux:
source django_env/bin/activate

# Instalar Django
pip install django

# Verificar instalación
django-admin --version
```
